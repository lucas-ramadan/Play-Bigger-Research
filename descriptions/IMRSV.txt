IMRSV is a technology company using computer vision for real-world, real-time data analysis. IMRSV's Cara software is a breakthrough in perceptive computing. Cara turns any webcam into an intelligent sensor enabling new methods of interacting with objects and spaces. Using a basic webcam, anyone can anonymously measure audience gender, age, attention time and glances in real world environments. Use cases include adaptive advertising, retail analytics and internet of things applications. Based in New York City with offices in Singapore, the company is privately backed by various investors.
