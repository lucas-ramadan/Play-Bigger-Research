Preact is a cloud-based customer success service that helps subscription software companies maximize customer lifetime value by reducing churn, acquiring new paid customers, and increasing user revenues.  Preact provides health metrics tailored to each user and account by tracking detailed session-level usage of web and mobile applications and server APIs. Preact predicts and informs which customers are likely to churn, renew, or upgrade by modeling usage and behavior patterns using big data analytics, behavioral science, and machine learning. In addition, Preact provides deep, real-time visibility into product usage to dramatically improve customer support and product quality.  Preact integrates with Gmail, Salesforce, Stripe, Zendesk, Desk.com, Helpscout, Mixpanel, and Marketo, and other leading cloud-based apps.

Founded in 2012, Preact is based in San Francisco.
